一 常见:标签分类

  • block: div h* ul li dt dd p

    独占一行,可以设置width,height,margin和padding属性

  • inline: span u a em b i u em

    宽度随元素的内容而变化,水平方向的margin和padding属性有效果,垂直没效果

  • inline-block: input img button texterea label

    表现inline,还可以设置width,height,margin和padding属性,但之间会有空白缝隙

  • 技巧:

  1. 文字类标签(p,h*,dt)内不放块级元素
  2. a里不放a,但可以放block元素

二 选择器分类

选择器 {属性key:value;属性key:value;}

  • 标签选择器(常用)
  • 类选择器(.class 常用)
  • id选择器(#)

  • 通配符选择器(*)

  • 后代选择器 (空格)

  • 子代选择器(>)
  • 交集选择器(e.class e#id)

  • 伪类选择器(:link(未点) :hover(移上去) :active(按下) :visited(点击后))

  • 并集选择器(,)

三 字体样式

单位: px(像素) em(本文字体大小)
font-style: normal /*italic*/
font-weight: 700 /*300*/
font-size: 16px
font-family: “microsoft yahei”, Arial;
font:font-style font-weight *font-size *font-family /*连写*/

四 文本样式

color: red /*#fff #fafafa rgb(255,0,0)*/
line-height: 33px /*顶 中 基 底*/

text-align: left /*right center 文字\行内元素\行内块居中 */
text-indent: 2em /*首行缩进2个字*/
text-decoration: none /*underline overline line-through*/
技巧: li去除默认前面点的方法:list-style:none

五 背景

background-color: black /*透明参数:rgba(0,0,0,.3)*/
background-image: url(1.jpg)
background-repeat: no-repeat /*repeat repeat-x repeat-y*/
background-attachment: scroll /*fixed*/
background-position: *x y /*top button right left px,只写一个时另一个默认center*/
background: background-color background-image background-repeat background-attachment background-position /*连写*/

技巧:

  1. 用于小图标(比如无序列表前面的小图标\搜索栏的小图片中)
  2. 用于局部背景
  3. 用于精灵图

六 CSS三大特性

  1. CSS层叠性(后者覆盖前者)
  2. 继承性(color,font-*,line-*,text-*)

  3. 优先级
    权重:
    无穷大=!important /*div {color:blue!important}*/
    1.0.0.0=行内样式
    0.1.0.0=id
    0.0.1.0=class,伪类
    0.0.0.1=标签
    0.0.0.0=*,继承样式

    七 盒子模型

  4. border
    border-width:1px
    border-style:solid /*hidden\solid\lodash*/
    border-color:red
    border: *border-width *border-style *border-color /*连写*/
    border-left: border-right: border-top: border-button:
    border-radius: 50%
    技巧: border:0 /*去除所有边框*/

  5. padding
    padding:上 右 下 左 /*1-上右下左 2-上下 右左 3-上下 右 左, 值:px,auto(自动充满),方位*/
    padding-*方位:
    技巧:
    1.padding会撑开带有width\height的盒子(width,height配置的是内容区域)
  6. margin
    margin:上 右 下 左 /*1-上右下左 2-上下 右左 3-上下 右 左, 值:px,auto(自动充满),方位*/
    margin-*方位:
    技巧:
    1.盒子居中:margin:0 auto
    2.外边距合并(以大为准,并不是相加的关系)
    3.外边距塌陷BUG(子盒子加margin-top,解决办法:1.父加padding-top 2.父加border-top 3.父加overflow:hidden)
    4.元素实际大小:content + padding + border
  7. 盒子阴影
    box-shadow: *h-shadow *v-shadow blur(虚实程度) spread(影子大小) color inset

八 浮动

css定位机制:普通流(标准流),浮动,定位
float:none /* left right */
核心:浮于上方,不占标准流空间,用于横向布局
技巧:
1.和父标签搭配使用,以父标签内容内浮动
2.一行全浮
3.浮动后,元素附加display:inline-block属性.
4.不完全脱离,底下如有文字图片会受影响.
清除浮动:
1.给父标签设置固定高度.
2.额外标签法(子元素最后加一个空标签)

3.父元素添加overflow属性 /*hidden auto scroll*/
4.伪元素(给标签添加class属性即可)

九 定位

定位:边偏移(top botton left right)+定位模式(position)
position:static /* relative absolute fixed*/
核心:
1.除static外,其他完全脱离文档流.
2.relative占茅坑,相对于自身定位.
3.absolute不占坑,相对于就近定位.
4.fixed,不占坑,相对于浏览器.
5.z-index:数字,控制叠放顺序,值越大越前.
技巧:
1.子绝父相
2.定位盒子居中:left:50%;margin-left:-自身宽度/2

十 秘籍

①居中汇总:
文字\行内元素\行内块
水平居中:text-align
垂直居中:line-heght=父标签高度
盒子居中:margin:0 auto
定位盒子居中:left:50%;margin-left:-自身宽度/2

③显示与隐藏:

  1. display(隐藏后,不占坑)
    display: none(隐藏) block(显示)
  2. visibility(隐藏后,占茅坑)
    visibility:hidden(隐藏) visible(显示)
  3. overflow(溢出隐藏-文字\行内元素\行内块)
    overflow: visible(默认) hidden(隐藏) scroll(滚动) auto(visible\scroll)
